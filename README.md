Static-web-generator for mooc courses from video libraries.

## Usage
```bash
python webgen.py
```
File directory should be
```
VideoLib
|--videos
    |--course_name
          | -- lesson_name
                  | -- video file
| --webgen.py

```

Output files 
```
VideoLib
|--videos
|--posters
|--srt
|--html
|--index.html

```
